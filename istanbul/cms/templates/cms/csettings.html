Adding or editing a user

Work with users straight from the admin menu.
  • Add a user: press top-right "ADD USER +". Then add the username (plain, lower case, no spaces or other symbols). Add a password (have the user change it later on), and Save it. Next fill in first name, last name, email address. Grant "Staff user" permission. Then make sure that the "Chosen groups" have istanbul_user.
  • Editing rights: Find the user in the list, and click it in order to edit its settings. Next, make sure to add istanbul_editor, so as to grant editing rights to the user. Should editing rights need to be revoked, double click the istanbul_editor, so that it moves out of the "chosen groups" list.
 

Lijst met hulp voor editors

Add or edit help for editors by going to the Editor help list. Select either the existing "Editor's handbook'" to edit the contents of that section. Or add a new section for the "Editor help" page.
