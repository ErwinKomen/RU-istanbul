Order images

The order of the images that belong to an installation can be changed this way:

  • Locate the installation and go to its details view.
  • Click the button Picture order... at the bottom.
  • Scroll down, order images by drag-and-drop (using tghe 'order' column).
  • Press the red button Save to fix the order.
  • Use the Cancel button to return to the original order.
To verify whether the process has been successful, re-load the page and check the current image order.
